# App icons & splash (placeholders)

These are solid-color placeholders so `eas build` succeeds. Replace with final brand assets:
- `icon.png` — 1024×1024, no transparency (App Store icon).
- `adaptive-icon.png` — 1024×1024 foreground (Android).
- `splash.png` — portrait launch image.
- `favicon.png` — web favicon.

Generate final assets from a designed source, or with `npx expo-optimize`.
