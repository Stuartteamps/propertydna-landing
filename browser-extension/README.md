# PropertyDNA Browser Extension

Turns any listing you're viewing (Zillow, Redfin, Realtor.com, FlexMLS, Homes.com)
into an instant PropertyDNA verdict — "overpriced by X%" — powered by our own AVM.

**Compliant by design:** it reads only the factual listing (address/price/beds/
baths/sqft) on the page *you* are already viewing, at *your* click, and opens our
own /price-check tool prefilled. No background scraping, no mass harvesting, no
credentials. Facts (address, price) aren't copyrightable; comps come from OUR data.

## Load it (unpacked — for you + testers today)
1. Chrome → `chrome://extensions`
2. Toggle **Developer mode** (top right)
3. **Load unpacked** → select this `browser-extension/` folder
4. Visit any Zillow/Redfin/MLS listing → click the gold **"🧬 Overpriced? — PropertyDNA"** button (bottom-right) or the toolbar icon → our verdict opens in a new tab.
